# hsia-npx-card
## Intro
It's a small project from Becode Liege in April 2021, to make a personal card shown in terminal.

## How to install it?
Type this command to install
<br/>
`npm i hsia-npx-card`

## How to use it
Type this command to get the card
<br/>
`npx hsia`
